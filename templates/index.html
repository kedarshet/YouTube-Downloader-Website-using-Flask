About Website

YouTube is an American online video sharing and social media platform launched by Steve Chen, Chad Hurley, and Jawed Karim in February 2005. Around the world, its users watch more than one billion hours of videos each day.

The Youtube downloader project is a python project. The object of this project is to download any type of video in a fast and easy way from youtube in your device.

This Website is basically a part of project. I have user Flask library to build Website & Pytube of backend downloading with HTML and CSS as frontend.
